# Sécurité

## Modèle de données

Mini Compta est une application **100% locale** : aucune donnée n'est transmise à un serveur externe. Toutes les écritures comptables sont stockées dans une base SwiftData sur l'appareil, protégée par le chiffrement iOS standard (Data Protection).

## Signaler une vulnérabilité

Pour signaler un problème de sécurité, ouvrez une issue avec le label `security` en décrivant :

1. La nature du problème
2. Les étapes pour le reproduire
3. L'impact potentiel

Les problèmes de sécurité seront traités en priorité.
